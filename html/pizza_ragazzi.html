Webbasiertes Computerspiel

Implementierung eines webbasierten Computerspiels zum Pizzabacken

Project Uni-Projekt in 5er Gruppe
Task Develop a web-based educational computer game, that familiarizes the user with a specific task
Duration 6 Months
Experiences HTML, JavaScript, Java, SQL, Git

"Pizza Ragazzi" in an educational computer game, to familiarize the player with the basic steps of baking pizza. The player has to assemble different types of pizza and correctly deliver time constrained orders.

From a technical standpoint, the game includes many Interfaces. The frontend of the game, written in HTML and JavaScrips, communicates with the Java-Backend via http-Requests. Additionally, the backend accesses an SQL-Database, that saves important information like user login information or Highscores.
