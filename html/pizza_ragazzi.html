Pizza Ragazzi

Implementierung eines webbasierten Computerspiels

Projekt Uni-Projekt in 5er Gruppe
Aufgabe Erstelle ein webbasiertes Lernspiel, mit dem bestimmte Tätigkeiten spielerisch nähergebracht werden
Dauer 6 Monate
Erfahrungen HTML, JavaScript, Java, SQL, Git

„Pizza Ragazzi“ ist ein Lernspiel, bei dem dem/der Spielenden das Pizza backen spielerisch nähergebracht wird. Es müssen unter Zeitdruck unterschiedliche Pizza-Bestellungen zubereitet und abgeliefert werden.

Aus technischer Sicht bedient das Projekt mehrere Schnittstellen. Das Frontend des Spiels, geschrieben mit HTML und JavaScript, kommuniziert über http-Requests mit dem Java-Backend. Außerdem greift das Backend auf eine SQL Datenbank zu, die Daten wie Spielerinformationen und Highscores speichert.
